Duck Mail
=========

Privacy e-mail service fully webbased, build by some DuckDuckGo community users.
Designed for 100% privacy e-mail, no tracking with advertisements or anything.

Fully secure, search information right from the box with the help from the DuckDuckGo API.
Based upon RoundCube Webmail.


Startup
=========
We need to start from the ground up, based upon Roundcube Webmail we will rewrite the code for support:

S/MIME
PGP
SSL online
Encrypted user database

Contribute
=========
You can create a PULL request to let your code be reviewd and be used for the Duck Mail.
A full opensource e-mail service.
